Morningstar 金融客戶 UI/UX 重建專案

🎯 3 秒亮點
在限定時間內完成多套金融服務界面重建，服務多家香港金融機構。

專案成果對比

項目挑戰解決方案
交付規模13 套系統（9 前台 + 4 後台）建立模組化設計系統，提升效率
時間壓力6 個月緊迫期限優先級管理 + 2 套系統轉交策略
技術複雜度4 種不同界面系統需重建統一設計語言 + 組件化設計方案
客戶影響19 個金融機構零中斷遷移段階式重建 + 測試流程

專案背景

  • 我的角色：Product Designer，負責整體界面設計與團隊協作
  • 專案時間：2024年5月開始
  • 核心客戶：香港主要上市銀行、指標保險集團、國際金融機構
  • 設計挑戰：將四種不同的金融服務界面，統一重建為現代化的網頁應用程式

設計挑戰與解決策略

核心挑戰 1：多樣化的界面需求

問題：多個客戶使用 4 種完全不同的界面系統，每種都有獨特的設計語言和用戶流程

解決方案：建立統一的設計系統

  • 設計可重用的金融 UI 組件庫
  • 建立一致的互動模式和視覺語言
  • 確保多語言界面（英文、繁中、簡中）的一致性

核心挑戰 2：金融級功能模組設計

需要重建的核心功能包括：

  • Screener：基金篩選工具界面
  • Compare：產品比較功能
  • Report：報告生成界面
  • X-Ray：投資組合分析工具
  • Portfolio Comparison：投資組合比較
  • Custom Admin Tool：客製化管理後台

設計策略：

  • 分析現有系統的用戶流程和互動模式
  • 重新設計符合現代 UX 標準的界面
  • 優化複雜金融數據的視覺呈現
  • 簡化管理後台的操作流程

核心挑戰 3：資源配置與工作分配

挑戰：在有限時間內完成大量界面重建工作

  • 需要設計多套不同的金融服務界面
  • 平衡設計品質與交付效率

策略：

  • 建立標準化的設計流程和組件庫
  • 專注於核心系統的設計優化

設計系統與解決方案

統一設計語言建立

  • 視覺系統：建立以數據為中心的設計語言，統一色彩系統，支援多品牌客製化
  • 互動模式：標準化表格排序和篩選模式、統一彈窗和提示系統、一致的導航和麵包屑設計、響應式設計適配不同螢幕尺寸

核心組件設計

  • Screener 基金篩選器：重新設計多維度篩選界面，優化篩選條件的視覺層級
  • Compare 產品比較：設計直觀的並排比較界面
  • Portfolio Comparison 投資組合比較：重新設計資產配置視覺化，優化投資組合分析的數據呈現，簡化組合調整的互動流程

設計過程

  • 分析現有的多個客戶界面系統和需求
  • 建立統一的設計規範和可重用組件
  • 進行前台和後台系統的界面重建

工作重點：

  • 確保每個客戶的品牌識別在新界面中得到保留
  • 統一但可客製化的設計語言建立
  • 核心功能模組的用戶體驗優化

專案價值與成果

  • 統一的金融界面視覺語言
  • 可重用的 UI 組件庫
  • 支援多品牌客製化的設計框架
  • 標準化的設計交付文檔

反思與學習

專案經驗與學習

  • 大規模界面重建經驗：負責多套金融服務界面的設計工作，學習如何在大型專案中建立一致性

專業成長

  • 金融界面設計專業：深入了解金融產品的界面需求和用戶使用情境
  • 設計系統建立：學會建立可擴展的設計規範和組件庫
  • 專案管理能力：在時間壓力下合理分配設計資源和工作優先級
